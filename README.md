# 🦷 React Calendar Widget for Dentist

A modern and responsive React Calendar Widget built for managing dentist appointments. This project was developed using **React + Vite** and follows the assignment requirements, including appointment scheduling, validation, loading states, accessibility considerations, and local data persistence.

## ⚙ Installation

Clone the repository

```bash
git clone https://github.com/your-username/react-calendar-widget-dentist.git
```

Go to project folder

```bash
cd react-calendar-widget-dentist
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```
